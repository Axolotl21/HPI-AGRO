Backend Developer

Description

  1. Participate in new product or new feature design.
  2. Adapt to changes and work within fixed tight deadlines.
  3. Work with own initiative to solve challenging problems and concepts.
  4. Confident in rationalizing with decisions.
  5. Participate in code and design reviews
  6. Communicate well with team (designers, developers and management)
  7. Promote usability best practices
  8. Explore new technologies

Requirement

  1. At least 5 years of experience in IT and 3 years of working experience using Golang.
  2. Familiar with Git workflow.
  3. Familiar with Kubernets/Microservice.
  4. Able to setup infrastructure
  5. Have strong logic
  6. Able to solve your own technical problem
  7. Contributions to OpenSource projects on GitHub is a plus
  8. Willing to learn new technology
